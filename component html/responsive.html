Avatar

Avatar is a kind of image which displays passport size photos in the form of a round image. For this first we need to add HTML where image is set with the help of image tag. After that in CSS just set the border-radius, width and height to 50%. "avatar" class and its done your avatar will be ready.

Avatar can be made of different sizes. Moreover, it is the component of CSS which can become responsive also. avatar (e.g. - "avatar medium" =medium sized avatar) class to switch to different sizes.
